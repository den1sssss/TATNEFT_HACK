### Installation & Run
Clone the repo and change to the project root directory:
```
git clone https://github.com/den1sssss/TATNEFT_HACK.git
cd TATNEFT_HACK
```

Install requirements:
```
pip install -r requirements.txt
```

And run:
```
streamlit run StreamLit.py
```
